# simsamples
Sample programs - SDL and simple simulations

## Compilation

```bash
mkdir build
cd build
cmake ../
```
## Alternative compilation procedure

This is example for "hello world"
```bash
cmake -B 01_hello_sdl_build -s 01_hello_sdl
cmake --build 01_hello_sdl_build
```
